# LilLearner Android Test Checklist

## App Launch
- [ ] App launches without crash and shows Home screen
- [ ] Home screen displays student name and day card
- [ ] Categories grid shows (Reading Lessons, Math Lessons, Language Arts, Printing)

## Navigation Tabs
- [ ] Tap "Progress" tab — Progress screen loads with content
- [ ] Tap "Reports" tab — Reports screen loads with content
- [ ] Tap "Profile" tab — Profile screen loads with content
- [ ] Tap "Home" tab — Returns to Home screen

## Home Screen Features
- [ ] Tap "Curricula" button — Curricula screen loads
- [ ] Navigate back to Home from Curricula
- [ ] Tap "Quick Log" button — Quick Log UI appears

## Authentication
- [ ] Tap "Profile" tab and check if sign-in or profile info is shown
