## The Pattern

Add this to the bottom of your CLAUDE.md:

> After any corrections from me, add an entry to `tasks/lessons.md`.
> Format: `date | what went wrong | rule for next time`
> Read `tasks/lessons.md` at the start of every session.
> Apply these rules before touching any code.

## How It Works

1. Every time you correct Claude Code, it writes the rule down
2. Next session, it reads all rules back and applies them before doing anything
3. After a month, your agent knows your patterns, preferences, and standards — built from real sessions
