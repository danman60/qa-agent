## The Stack (Work Backwards)

If you're feeling unmotivated:

| If You Don't Have... | Build... |
|---------------------|----------|
| Confidence | Evidence |
| Evidence | Skills |
| Skills | Discipline |
| Discipline | Reframe — you already have it in other domains |

> "You have the core skill. Apply it to a new domain and then give time, time."

## Full Transcript

If you're feeling unmotivated at the end of the year, hear me out. If you don't have confidence, build evidence. If you don't have evidence, build skills. If you don't have skills, build discipline. And if you don't have discipline, you need to reframe the things you've done in the past — realize that you have that discipline, just in different domains. So you have the core skill. You should apply it to a new domain and then give time, time.
