# Sysadmin — Claude Code Skills Gotcha List

**Source:** https://www.facebook.com/share/r/18GJnofFXd/
**Transcribed:** 2026-03-22

## The Pattern

Build a "gotcha list" in every Claude Code skill. Every time Claude Code makes a mistake inside a skill, add one line to the bottom of the skill's gotcha section:

> What went wrong, and the rule to follow next time.

Claude reads it on every activation. The same mistake never happens more than once.

**A skill without a gotcha list is just a prompt. A skill with one gets better every single session.**
